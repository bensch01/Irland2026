# Irland Reiseplanung Website

Öffnen:
- `index.html` im Browser öffnen.
- Für Karte und Live-Wetter braucht die Seite Internet.
- Die Website liest `reise-data.json`; bitte beide Dateien im gleichen Ordner lassen.

Bearbeiten:
- Auf „Bearbeiten“ klicken.
- Titel, Intro, Tagesüberschriften, Hotelzeilen und Tagespläne direkt auf der Seite ändern.
- Mit „Änderungen speichern“ wird lokal im Browser gespeichert.

Mit Freunden teilen:
- Den kompletten Ordner oder die ZIP-Datei verschicken.
- Noch besser: Ordner bei Netlify Drop, GitHub Pages oder einem Webspace hochladen und den Link teilen.

Quellen:
- Reiseinhalt: Reisen.xlsx, Reiter „Irland“ und „Ring of Kerry“
- Karte: OpenStreetMap/Leaflet
- Wetter: Open-Meteo API
- Sehenswürdigkeiten: Wikipedia-Suche
